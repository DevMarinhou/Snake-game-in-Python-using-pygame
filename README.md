Jogo da Cobrinha
Um clássico jogo da cobrinha desenvolvido em Python com o uso da biblioteca Pygame.

Funcionalidades

Interface simples e intuitiva

Pontuação crescente a cada alimento consumido

A cobra aumenta de tamanho ao comer

Game over com exibição da pontuação final

Reinício rápido com a tecla Espaço

Velocidade progressiva (opcional, dependendo da implementação)

Como jogar

Tecla	Ação
⬆️ Seta para cima	Move para cima
⬇️ Seta para baixo	Move para baixo
➡️ Seta para direita	Move para direita
⬅️ Seta para esquerda	Move para esquerda

Espaço Reiniciar o jogo
ESC para Sair do jogo

Requisitos
Python 3.7 ou superior

Pygame
